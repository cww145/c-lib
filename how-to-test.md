# c-lib

## Visual Studio Code での実行

### 拡張機能

- ms-vscode.cmake-tools

### 全テストを実行する場合

1. `Shift + Comand + P` を押す
2. `Cmake: Run tests` を選択

### 1つのテストを実行する場合

1. `Shift + Comand + P` を押す
2. `Cmake: Build Target` を選択
3. 対象のテストを実行

### 入出力を手入力する場合

1. `Shift + Comand + P` を押す
2. `Cmake: Run Without Debugging` を選択
